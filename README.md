### Создать виртуальное окружение и сразу запустить его:
```
python -m venv venv
venv\Scripts\activate
```

### Установить все нужные пакеты:
```
pip install -r requirements.txt
```

### Указать основной файл:
```
set FLASK_APP=cipher.py
```

### Оживить БД:
```
flask db upgrade
```

### Запуск:
```
flask run
```
